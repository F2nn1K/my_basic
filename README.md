# MY BASIC - Catálogo Online

Catálogo digital da **MY BASIC**, loja de roupas femininas com foco em peças básicas, elegantes e versáteis.

## Sobre

Site estático de catálogo de produtos com design delicado em tons rosé, integração com WhatsApp para pedidos e layout 100% responsivo.

## Tecnologias

- HTML5
- CSS3 (animações, grid, flexbox, backdrop-filter)
- JavaScript (filtros, modal, scroll animations)
- Google Fonts (Cormorant Garamond + Jost)

## Funcionalidades

- Catálogo de produtos com fotos
- Filtro por categoria (Regatas / Bodies)
- Modal de detalhes do produto
- Botão de pedido via WhatsApp
- Menu responsivo para mobile
- Animações suaves de scroll

## Deploy

Hospedado na [Netlify](https://www.netlify.com/) com deploy direto deste repositório.
